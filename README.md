# Notes (Reactjs)

Essa aplicação foi desenvolvida durante o NLW Experts da Rocketseat utilizando React, TypeScript, Tailwind e a SpeechRecognition API.

## Requisitos

- Node.js;

## Tecnologias

- React.js;
- Vite.js;
- TypeScript;
- Lucide React;
- Tailwindcss;
- Date-fns;

## Steps

- Clone o repositório;
- Instale as dependências (`npm install`);
- Rode a aplicação (`npm run dev`);
